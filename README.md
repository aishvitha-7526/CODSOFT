# CODSOFT Web Development Internship

This repository contains the projects completed as part of the CodSoft Web Development Internship program.

## Projects

### 1. Personal Portfolio Website
A personal portfolio website developed using HTML and CSS to showcase my profile, skills, projects, resume, and contact information.

**Technologies Used:**
- HTML5
- CSS3

### 2. Landing Page
A responsive landing page designed using HTML and CSS with a navigation bar, hero section, features section, and footer.

**Technologies Used:**
- HTML5
- CSS3

### 3. Calculator
A simple calculator application developed using HTML, CSS, and JavaScript that performs basic arithmetic operations.

**Technologies Used:**
- HTML5
- CSS3
- JavaScript
